LearnSocket
==========

项目为学习socket网络编程中的一些练习,将包括以下部分:

基础部分
----------

###simple_server
基本api构建的简单服务器

###select_server
多路复用api(select)构建的select服务器

###poll_server
多路复用api(poll)构建的poll服务器

###epoll_server
多路复用api(epoll)构建的epoll服务器

实战部分
----------
###non_blocking_connect
非阻塞式connect

###chat_server
聊天程序

###tcp_udp_sever
同时处理tcp udp

###xinetd
超级服务xinetd
